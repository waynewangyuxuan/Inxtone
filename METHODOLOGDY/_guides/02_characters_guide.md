## 一、人物内核设计

### 1.1 五大内在矛盾类型（来自B20 2.1）

每个有深度的人物都应有核心矛盾：

| 类型 | 描述 | 常见表现 |
|------|------|----------|
| **欲望 vs 道德** | 本能与规范的冲突 | 复仇vs宽恕、权力vs良知 |
| **理想 vs 现实** | 应然与实然的碰撞 | 正义信念vs灰色地带 |
| **自我 vs 社会** | 个体与群体的张力 | 反叛vs归属、真我vs面具 |
| **爱 vs 责任** | 情感与义务的冲突 | 儿女情长vs家国天下 |
| **生存 vs 尊严** | 生存本能与人格的冲突 | 卑微求生vs宁死不屈 |

**选择建议**:
- 主角应有主要矛盾 + 次要矛盾
- 核心配角应有明确的主要矛盾
- 矛盾应随剧情发展产生变化

### 1.2 三层动机模型（来自B20 2.2）

```
表层动机（可见的行动目标）
    ↓
深层动机（隐藏的心理驱动）
    ↓
核心动机（无意识的根源需求）
```

| 层次 | 定义 | 揭示时机 |
|------|------|----------|
| 表层 | 角色明确说出的目标 | 开篇即揭示 |
| 深层 | 角色可能意识到但不愿承认 | 中期通过冲突揭示 |
| 核心 | 角色自己也未意识到 | 高潮或结局揭示 |

**示例**:

| 角色 | 表层 | 深层 | 核心 |
|------|------|------|------|
| 复仇者 | 为师报仇 | 证明自己的价值 | 渴望被认可 |
| 天才少年 | 成为最强 | 超越父亲阴影 | 渴望父亲的爱 |
| 守护者 | 保护家人 | 弥补过去的遗憾 | 恐惧失去 |

### 1.3 八大人物模板（来自B20 2.5）

| 模板 | 核心特征 | 弧光方向 | 网文适配 |
|------|----------|----------|----------|
| **复仇者** | 被亏欠，追求清算 | 放下仇恨 或 被仇恨吞噬 | 废柴逆袭开局 |
| **守护者** | 保护他人/信念 | 学会放手 或 牺牲成全 | 护短型主角 |
| **追寻者** | 寻找真相/意义 | 发现真相 或 与未知和解 | 探索型故事 |
| **反叛者** | 挑战秩序/权威 | 建立新秩序 或 理解旧秩序 | 反套路主角 |
| **救赎者** | 弥补过去错误 | 获得救赎 或 赎罪失败 | 有过去的主角 |
| **旁观者** | 不介入/中立 | 被迫选择立场 | 咸鱼被迫营业 |
| **牺牲者** | 为更大目标舍弃 | 牺牲完成 或 找到另一条路 | 悲剧美学 |
| **堕落者** | 从光明走向黑暗 | 堕落完成 或 悬崖勒马 | 反派、黑化 |

## 二、Wayne原则在人物设计中的应用

### 2.1 R1检查：禁止"养狗式"人物关系

**设计配角时必问**:

| 问题 | 期望回答 |
|------|----------|
| 这个配角除了"忠于主角"还有什么追求？ | 有明确的个人目标 |
| 如果主角做了错误决定，配角会反对吗？ | 会，在某些情况下 |
| 配角的人生目标是否独立于主角？ | 是的，有自己的人生 |
| 配角为什么跟随主角？ | 有具体原因，不是"被魅力折服" |

**健康的配角关系模式**:

| 关系类型 | 描述 | 示例 |
|----------|------|------|
| 共同目标 | 暂时同路，目标一致时合作 | 同门师兄弟各有志向 |
| 互惠利益 | 各取所需，利益交换 | 主角需要信息，配角需要保护 |
| 情感羁绊 | 有发展过程的真实情感 | 从敌对到理解到信任 |
| 价值认同 | 认同理念而非人格依附 | 认同主角的选择而非崇拜主角 |

### 2.2 R2检查：禁止角色工具化/符号化

**多面性构建**（来自B20 2.3）:

每个角色至少应有2-3个不同面向：

```markdown
## 角色多面性

### 公开面
[在大多数人面前展现的样子]

### 私密面
[只在亲近人面前展现的样子]

### 隐藏面
[连自己都未完全意识到的一面]

### 压力下的面
[极端情况下暴露的真实反应]
```

**检验方法**:
- 能否用一个标签完全概括这个角色？（如果能 → 有问题）
- 这个角色的行为是否完全可预测？（如果是 → 有问题）
- 换一个人来扮演这个"功能"，有区别吗？（如果没有 → 有问题）

### 2.3 P1实践：尊重每一个角色

即使只出场20章的角色，也应该有：

- **自己的目标**（哪怕很小）
- **独特的记忆点**（外貌、口头禅、行为习惯）
- **有尊严的退场**（不必死亡，但要有交代）

**设计时问自己**: "如果这个人是一本书的主角，他的故事是什么？"

### 2.4 P2实践：每个人都有自己的想法

**配角设计顺序**:
1. 先定义"他自己是谁"
2. 再定义"他和主角的关系"

**配角应该有**:
- 自己的判断（可以和主角不同）
- 独立于主角的人际关系
- 自己的困境和追求

## 三、关系网络设计

### 3.1 核心关系类型（来自B20第四部分）

| 类型 | 特征 | 张力来源 |
|------|------|----------|
| **同行者** | 目标一致的伙伴 | 方式分歧、资源竞争 |
| **对手** | 良性竞争关系 | 相互超越、观念碰撞 |
| **敌人** | 目标冲突 | 生死对立、价值观冲突 |
| **导师** | 引导成长 | 观念冲突、超越与放手 |
| **知己** | 深度理解 | 期待与失望 |
| **爱人** | 情感羁绊 | 责任与爱、成长与陪伴 |

### 3.2 关系矩阵设计

用矩阵可视化人物关系：

```
        主角    配角A   配角B   反派
主角    -       同行者  知己    敌人
配角A   同行者  -       暗恋    旧仇
配角B   知己    被暗恋  -       不知
反派    敌人    旧仇    不知    -
```

**设计原则**:
- 主要人物之间应有直接关系
- 关系应有发展空间（不是一成不变）
- 关系应产生剧情张力

### 3.3 非依附关系模板（来自B20 4.2）

设计配角与主角关系时，使用此模板确保不陷入"养狗"：

```markdown
## [配角]与主角的关系

### 关系性质
[同行者/对手/知己/...]

### 配角加入主角的原因
[具体事件，不是"被魅力折服"]

### 配角的个人目标
[独立于主角的追求]

### 配角会反对主角的情况
[在什么事情上会意见不同？]

### 配角可能离开主角的情况
[什么情况下这段关系会破裂？]

### 主角需要配角的什么
[主角从这段关系中获得什么？]
```

## 五、B20检查清单（来自B20第七部分）

### 5.1 单个人物评分

| 维度 | 检查项 | 分数(0-10) |
|------|--------|------------|
| 内核 | 是否有清晰的核心矛盾？ | |
| 动机 | 三层动机是否完整？ | |
| 多面性 | 是否有多个面向？ | |
| 一致性 | 行为是否符合人设？ | |
| 独特性 | 是否有独特记忆点？ | |
| 弧光 | 是否有成长/变化？ | |
| | **总分（/60）** | |

**评分标准**:
- 50+ 分: 优秀，可作为核心人物
- 40-50: 良好，需微调
- 30-40: 及格，需要加强某些维度
- 30以下: 需要重新设计

### 5.2 关系网络评分

| 维度 | 检查项 | 分数(0-10) |
|------|--------|------------|
| 张力 | 核心人物之间是否有张力？ | |
| 多样性 | 关系类型是否多样？ | |
| 发展 | 关系是否有发展空间？ | |
| 非依附 | 是否避免了"养狗"？ | |
| | **总分（/40）** | |

**评分标准**:
- 35+ 分: 优秀
- 28-35: 良好
- 20-28: 需要改进
- 20以下: 关系网络设计有问题
