## 伏笔管理原则

1. **埋设时要自然**: 不能太刻意
2. **回收时要惊喜**: 读者应该"啊，原来如此"
3. **有计划地管理**: 不能埋了就忘
4. **适度即可**: 不必每个细节都是伏笔
5. **长短结合**: 短期(5-20章)、中期(20-100章)、长期(100章+)
