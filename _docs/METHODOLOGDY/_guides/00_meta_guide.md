## 一、核心创意设计

### 1.1 一句话概括

你的故事必须能用一句话吸引读者。这句话应包含：

**公式**: [主角身份] + [核心冲突/困境] + [独特卖点]

**示例**:
- 《诡秘之主》: "穿越者发现自己在神秘世界中扮演序列途径，需要在疯狂与力量之间寻找平衡"
- 《没钱修什么仙》: "一个穷鬼在修仙世界用现实主义视角吐槽一切修仙套路"
- 《剑来》: "一个泥瓶巷少年背负师父的剑，走出一条属于自己的道"

**检验标准**: 念给一个不看网文的朋友听，他是否会问"然后呢？"

### 1.2 目标读者画像

明确你的目标读者有助于后续所有决策：

| 维度 | 选项 | 你的选择 |
|------|------|----------|
| 年龄 | 15-25 / 25-35 / 35+ | |
| 经验 | 网文老鸟 / 有一定阅读量 / 新读者 | |
| 偏好 | 爽感为主 / 爽感+深度 / 深度为主 | |
| 时间 | 碎片阅读 / 深度阅读 | |
| 期待 | 爽文解压 / 情感共鸣 / 世界探索 | |

### 1.3 三个核心卖点

列出你的书最核心的三个吸引点：

1. **卖点一**: [是什么] — [为什么读者会喜欢]
2. **卖点二**: [是什么] — [为什么读者会喜欢]
3. **卖点三**: [是什么] — [为什么读者会喜欢]

**检验**: 这三个卖点是否和市面上的书有明显区别？

### 1.4 模式选择

根据FRAMEWORK_V3.0，网文有四种成功模式：

| 模式 | 核心驱动 | 代表作 | 适合情况 |
|------|----------|--------|----------|
| 世界观驱动型 | 独特有趣的世界设定 | 诡秘之主、咒禁山海 | 你有完整的世界观创意 |
| 人物驱动型 | 有魅力的角色群像 | 剑来、雪中悍刀行 | 你擅长写人物 |
| 爽点驱动型 | 持续的爽感节奏 | 斗破苍穹、大奉打更人 | 你追求商业表现 |
| 反套路驱动型 | 颠覆读者预期 | 没钱修什么仙、道诡 | 你想做创新 |

**你的选择**: ____________________

**注意**: 可以是融合型（如"世界观+人物"），但应有主次。

## 二、Wayne原则红线

### 2.1 绝对红线

在创作开始前，必须确认以下红线：

#### 🚫 R1: 禁止"养狗式"人物关系

**定义**: 配角对主角无条件崇拜、跪舔、无自我意识

**自检问题**:
- 你的配角除了"忠于主角"还有什么追求？
- 如果主角做了错误的决定，配角会反对吗？
- 配角的人生目标是否独立于主角？

**常见陷阱**:
- "主角救了他一命，所以他永远忠诚" — ❌ 这是交易，不是人格
- "他被主角的气质折服" — ❌ 太抽象，需要具体的互动理由

**替代方案**: 配角与主角的关系应基于：
- 共同的目标（暂时同路）
- 互惠的利益（各取所需）
- 真实的情感（有发展过程）
- 价值观认同（而非人格依附）

---

#### 🚫 R2: 禁止角色工具化/符号化

**定义**: 角色只是功能的载体，没有独立人格

**自检问题**:
- 能否用一个标签完全概括这个角色？（如果能，有问题）
- 这个角色的行为是否完全可预测？
- 换一个人来扮演这个"功能"，有区别吗？

**常见陷阱**:
- "军师型角色"只会献计 — ❌ 他不会有自己的野心？
- "忠犬型角色"只会保护 — ❌ 他不会有自己的恐惧？
- "毒舌型角色"只会吐槽 — ❌ 他的毒舌是否是一种防御机制？

**替代方案**: 使用B20的"多面性构建"：
- 至少给角色2-3个不同面向
- 让角色在不同情境下表现不同
- 揭示角色"功能"背后的原因

---

#### 🚫 R3: 禁止草菅人命

**定义**: 角色的死亡没有重量、没有意义、没有余波

**自检问题**:
- 如果这个死亡发生在主角身上，你会这样写吗？
- 这个角色死后，有人会真正悲伤吗？
- 这个死亡除了"证明反派残忍"还有什么作用？

**常见陷阱**:
- 为了让反派"够坏"而随意杀人 — ❌
- 龙套角色被秒杀没有任何后续 — ❌
- 死亡变成"升级需要的经验值" — ❌

**替代方案**: 使用B20的"死亡美学"：
- 重要角色的死亡需要铺垫和意义
- 即使是龙套，也可以用一两句话赋予人性
- 死亡应该产生余波（对其他角色的影响）

---

#### 🚫 R4: 禁止神话模型代替社会模型

**定义**: 强者拥有"神性"，不再受社会规则约束

**自检问题**:
- 你的世界里，强者是否"说什么都对"？
- 弱者是否有任何谈判筹码？
- 力量是否等于绝对正确？

**常见陷阱**:
- "我比你强，所以我说了算" — ❌ 这是野兽逻辑
- "实力为尊的世界" — ⚠️ 可以存在，但需要制衡
- 强者对弱者"赐予"而非"交换" — ❌

**替代方案**: 使用社会关系模型：
- 老板再强也需要员工
- 皇帝再强也需要朝臣
- 强者也有求于人的时候
- 弱者有强者需要的东西（信息、技能、忠诚、时间）

---

### 2.2 核心原则

在构思阶段，就应将这些原则嵌入设定：

#### ✅ P1: 尊重每一个角色

**即使只出场20章的角色，也应该有**:
- 自己的目标（哪怕很小）
- 独特的记忆点（外貌、口头禅、行为习惯）
- 有尊严的退场（不必死亡，但要有交代）

**构思阶段应用**: 当你设想核心配角时，问自己——"如果这个人是一本书的主角，他的故事是什么？"

#### ✅ P2: 每个人都有自己的想法

**配角不是NPC**:
- 有自己的判断
- 可以不同意主角
- 有独立于主角的人际关系

**构思阶段应用**: 在设计配角时，先不要定义"他和主角的关系"，先定义"他自己是谁"。

#### ✅ P3: 力量差距 ≠ 人格不平等

**强者可以更强，但**:
- 弱者仍有独特价值
- 对话可以平等进行
- 尊重不是靠力量换来的

**构思阶段应用**: 在设计力量体系时，思考——弱者能为强者提供什么？

#### ✅ P4: 使用社会关系模型

**参考现实中的权力关系**:
- 任何组织都有层级，但层级不等于奴役
- 服从有条件，忠诚有边界
- 强者需要弱者的某些东西

**构思阶段应用**: 在设计势力/门派时，思考——这个组织为什么不崩溃？成员为什么不离开？
